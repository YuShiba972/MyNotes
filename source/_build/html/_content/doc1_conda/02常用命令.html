2. conda常用命令速查

2.1. conda环境操作

查询环境 : conda info –envs | conda env list

激活环境 : conda activate 环境名

创建环境 : conda create –name 环境名 python==3.8

删除环境 : conda env remove –name 环境名

克隆环境 : conda create –clone 已存在的环境 –name 环境名

2.2. conda包操作

查看当前虚拟环境的包 : conda list

查看指定虚拟环境的包 : conda list –name 环境名

2.3. conda配置文件.condarc修改配置命令

conda环境安装根目录添加 : conda config –add envs_dirs /path/to/your/preferred/envs

conda包安装根目录添加 : conda config –add pkgs_dirs /path/to/your/preferred/pkgs

conda所有配置(包括默认) : conda config –show

conda显示配置文件 : conda config –show-sources

conda镜像地址显示 : conda config –set show_channel_urls yes

conda镜像源添加 : conda config –add channels 源地址
