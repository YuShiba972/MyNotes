05 代码块指令

备注

全局代码配置：highlight

设置文档中大于5行的python代码显示行数:

.. highlight:: python
  :linenothreshold: 5

设置文档所有python代码显示行数(force优先级大):

.. highlight:: python
  :force:

备注

代码块：使用code-block或sourcecode

shell代码块：

.. code-block:: shell

  echo 'hello world!'

python代码块：
.. sourcecode:: python

  def main():
    print('hello world')

shell代码块：

echo 'hello world!'

python代码块：

def main():
  print('hello world')

备注

代码块可选项：linenos行数显示 | emphasize-lines强调行 | caption导航文本 | name预定义引用名

.. code-block::
  :linenos:
  :emphasize-lines: 3,5
  :caption: this.py
  :name: this-py

  def some_function():
    interesting = False
    print('This line is highlighted.')
    print('This one is not...')
    print('...but this one is.')
this.py

4  print('This line is highlighted.')

备注

外部代码文件引用：literalinclude，可选项：encoding编码 | lines引用指定行 | …

.. literalinclude:: index.rst
  :linenos:
  :language: rst
  :emphasize-lines: 1-7,10
  :encoding: utf-8
  :lines: 1-11
